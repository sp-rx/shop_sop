Current Prescription

Ask what prescription they currently use. There are a few ways to get the info:
  1. Customer provides details (photo or documents)
  2. Use the lensmeter to check current glasses
    • Single vision: Center the lens and record sphere, cylinder, axis
    • Multifocal: Plot the zone, then check power of distance zone. Read engraving for Near power.
  3. If existing customer, check Gmail record by WhatsApp number or name
  4. If existing customer, check POS system record by WhatsApp number or name

New Prescription

Firstly, find out if a current prescription exist (it can be old but so long usable)
  1. If a current RX exists, use it as the base and proceed with Vector RX.
  2. If no current RX, do a base prescription using retinoscopy or a Snellen chart, then proceed with Vector RX.
  3. Vector RX can either be done as double eye or single eye, depending on the requirement of the test.

Eye Exam Types

There are 2 types of eye exam:
  1. Base Refraction:
    • A quick and simple estimation of prescription
    • Done using a Snellen chart or retinoscopy
    • Takes less than 2 minutes
    • Typically accurate up to about 90% vision
  2. Vector Refraction:
    • Refines the base prescription to achieve maximum accuracy
    • Our in-house specialty test, not available elsewhere
    • Can be done monocular (single eye) or binocular (both eyes)
    • Provides the most comfortable and best possible vision outcome

Based on situation:
  1. If a current RX exists, use it as the base and proceed with Vector RX.
  2. If no current RX, establish a base using retinoscopy or Snellen chart, then proceed with Vector RX.
  3. Every customer needs to go through Vector RX to determine the prescription when vision is maximised.
  4. But the final diagnosed prescription might be adjusted for special reasons depending on situations.

Contact Lenses Sales

There are 2 types of contact lens sale: Urgent and Not Urgent.
  • Urgent Sale, ready-stock only.
    1. Ask what prescription they currently use. There are few ways to get the info:
      • Customer provides details (photo or physical pack)
      • Use lensmeter if they are wearing glasses. Then use our app to convert the power to contact lens.
      • Perform a quick eye test.
      • If existing customer, check Gmail record by whatsapp number or name.
      • If existing customer, check POS System record by whatsapp number or name.
    2. Ask preferences:
      • Daily or Monthly disposable
      • Clear or Coloured
    3. Match to product & check stock:
      • Daily Clear Acuvue Oasys 1 day Hydraluxe
      • Daily Colour Acuvue Define for big pack
      • Daily Colour Seed Eye Coffret for small pack
      • Monthly Clear Acuvue Vita
      • Biweekly Clear Acuvue Oasys 2 weeks
      • Monthly Colour Bliss (high performance & comfort)
      • Monthly Colour Icoloris (basic performance & comfort)
    4. Use app to check price.
    5. Make Payment: Credit Card, QR, or Cash
    6. Send Receipt via whatsapp
    7. Create customer details in system, use our app to send:
      • Take photo
      • Send details via Gmail with price, product, notes

  • Not Urgent Sale
    1. Ask if customer has a preferred brand or type.
      • Customer can provide photos or details of their current lenses.
    2. Check if the requested product is in stock:
      • If yes → follow the Urgent Sale flow (ready-stock only).
      • If no → show product options via app and double-check product, quantity, and price.
    3. Ask what prescription they currently use. There are few ways to get the info:
      • Customer provides details (photo or physical pack)
      • Use lensmeter if they are wearing glasses. Then use our app to convert the power to contact lens.
      • Perform a quick eye test.
      • If existing customer, check Gmail record by whatsapp number or name.
      • If existing customer, check POS System record by whatsapp number or name.
    4. Take payment first to confirm the order (Credit Card, QR, or Cash).
    5. Record customer details in system:
      • Prescription, brand, quantity, price, and any notes.
    6. Proceed to order

Glasses Sales

  1. Understand why the customer wants to change glasses:
    • Badly made glasses
    • Scratched lenses
    • Vision getting worse
    • Dislike current frame design
    • Old glasses / time to change
  2. This determines whether prescription should be changed or maintained
    • If vision-related (badly made / vision worse) → allow more adjustment
    • If non-vision-related → maintain or adjust safely without disrupting current comfort
  3. Ask for current prescription:
    • Check existing glasses (lensmeter)
    • Check records (Gmail / POS)
    • If none → proceed to create the base prescription
  4. Select frame first before eye examination:
    • Fit, size, comfort
    • Face shape and preference
    • If customer wants eye exam first → inform that eye exam is chargeable if no glasses are made
  5. Proceed to eye examination.
    • Start with base or create the base
    • Then proceed with the vector RX

Glasses Fitting

After the eye examination, proceed to take fitting details.
  1. Take measurements:
    • PD (pupil distance, the horizontal distance between both eyes)
    • NOC (nosepiece to optical center, the vertical distance from the optical center of the lens, or the pupil to the center of the nosepiece)
  2. PD and NOC are critical for visual comfort.
    • Changes will affect how the customer perceives vision
    • Maintain as similar as possible to current glasses
    • Only adjust if there are complaints (e.g. headache, strain, dizziness when wearing the existing parameters)
After the fitting is taken, proceed to recommending ophthalmic lens to suit the budget and performance.
  1. Recommend lens using our Ophthalmic Lens app.
    • Key in the customer's prescription and the app will generate the recommendation.
  2. Finalize lens choice:
    • Lens type (single vision, anti-fatigue, multifocal)
    • Coating / options
  3. Confirm price and proceed to payment
